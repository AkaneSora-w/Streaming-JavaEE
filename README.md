# Streaming-JavaEE

Java EE demo project focused on distributed programming concepts applied to
a simple streaming subscription system.

## Technologies
- Java EE
- EJB (Stateless, Singleton)
- JPA
- JMS
- Web Services

## Project Scope
This project demonstrates how different Java EE components can be combined
to build a distributed application.

## Notes
The code is provided as-is and reflects an academic-style implementation.
No refactoring or further improvements have been applied.
